aokp-gerrit-theme
=================

Dark AOKP theme for Gerrit (built on v2.8-981-g553b64f-dirty)

Installation
====

Simple. Just copy the /etc and /static folders to the root of Gerrit. You may also want to backup anything that would be overwritten.
